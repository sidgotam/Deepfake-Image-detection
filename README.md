# Deepfake-Image-detection
CNN-based Deepfake Image Detection System with Flask API and interactive web interface for real-time fake image identification and visualization using Grad-CAM.
## 📊 Dataset

This project uses the "Deepfake and Real Images" dataset from Kaggle.

Dataset Link:
https://www.kaggle.com/datasets/manjilkarki/deepfake-and-real-images

The dataset contains real and AI-generated (deepfake) facial images used for training and evaluating the model.

Note: Dataset is not included in this repository due to size limitations.
